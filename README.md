# go-portal-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不针对特定域名或具体网站，可供查阅、测试或作为参考资料使用。

## 项目简介

`go-portal-notes` 是一个简单的静态页面存档仓库。仓库中的 HTML 文件彼此独立，不依赖后端服务或数据库。适合存放一些临时的、演示性质的页面，或用于记录特定场景下的界面快照。

## 目录结构

```
go-portal-notes/
├── README.md
├── index.html         # 仓库入口页（可选）
└── pages/             # 存放各独立 HTML 页面
    ├── page-1.html
    ├── page-2.html
    └── ...
```

- `index.html`：可选的仓库入口文件，用于快速浏览页面列表。
- `pages/`：所有独立的 HTML 页面存放于此目录，命名尽量体现内容或用途。

## 页面归档说明

- 每个 HTML 文件均为自包含的独立页面，可单独打开使用。
- 页面可能包含简单的样式和脚本，但不依赖外部资源（除非在页面中显式引用）。
- 归档目的是为了记录和分享页面形态，不保证所有页面长期可用或内容持续更新。

## 维护说明

- 本仓库由维护者不定期更新，添加新的页面或清理过时内容。
- 欢迎提交 Issue 或 Pull Request 来报告问题或建议改进。
- 仓库不承诺提供技术支持或即时响应。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/go-portal-notes.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 等服务部署为静态站点（需自行配置）。
